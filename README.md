# Тестовое задание от Reviro

## __ Создать приложение, которое позволяет пользователям искать и отображать текущую погоду и прогноз в их городе__.

## Инициализация веб-приложения
### Сборка пакетов. 
```js
npm ci
```
__или__
```js
yarn
```

### Запуск сервера, компиляция файлов.
```js
npm run dev
```
__или__
```js
yarn dev
```
